Music Genre Classification

I developed a music genre classifier as my final year project. For training I used the FMA medium dataset and with data augmentation the dataset size went upto 140 GB. Spectrograms were generated from the training data and then the spectrograms to a Convolution Neural Network. It took 40 hours to train and accuracy of 82% was achieved.

Pneumonia Detector

While working on the Corona X-Ray dataset I came across the Pneumonia dataset on Kaggle. I had used the normal lung images from this dataset in corona dataset to increase the amount of images for training. So I decided why not train a CNN model to detect if a patient has Pneumonia on the basis of X-ray. The dataset on Kaggle also does not have many images so with data augmentation around 5000 images where used while training.The project can be found on my Github.

Image Noise removal and image reconstruction using Auto Encoders.

The architecture of an autoencoder looks somewhat like an hourglass. I tried implementing autoencoder using Keras with the LFW dataset. An autoencoder is able to compress the information from first layer and reconstruct it back using the final layer. Another use of an autoencoder is to denoise an image. I introduced noise in the dataset and trained again and it was able to somewhat able to remove the noise too. Overall , learning about autoencoders was really interesting.
